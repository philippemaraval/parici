# Visual QA Checklist v1

## Marque
- [ ] Le bleu logo `#12297A` apparait sur les elements marque.
- [ ] Aucune teinte violette n'apparait dans les badges/profil.
- [ ] Les etats success/warning/danger sont semantiquement corrects.

## Layout et Typo
- [ ] Titres: `Montserrat`; texte: `Nunito`; chiffres score/chrono: `JetBrains Mono`.
- [ ] Contraste texte principal >= AA.
- [ ] Hierarchie visuelle stable entre panels.

## Composants
- [ ] Bouton primaire coherent (gradient mer).
- [ ] Bouton stop coherent (danger).
- [ ] Pills score/difficulte alignes sur palette.
- [ ] Inputs/selects avec bordures pierre + focus bleu.

## Carte
- [ ] Rues normales en jaune soleil.
- [ ] Correct/incorrect en vert/rouge.
- [ ] Overlay arrondissement en bleu marque.
- [ ] Monuments en palette mer.

## Profil / Progression
- [ ] Cartes profil, badges, avatars utilisent la meme famille visuelle.
- [ ] Etats locked/unlocked lisibles sans ambiguite.
- [ ] Leaderboard et profil partagent les memes couleurs de statut.

## Daily
- [ ] Table/hints/resultats respectent la palette officielle.
- [ ] Carte de partage Daily reprend la marque Paris.

## Responsive
- [ ] Mobile <= 900px: lisibilite score/chrono/boutons.
- [ ] Aucun debordement de panneau ou tooltip.
- [ ] Interactions tactiles lisibles (hover equivalent).
